# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-25
### Added
- Initial public release
- Probe request capture and analysis
- Real-time terminal output with color coding
- MAC address vendor lookup
- CSV and JSON export options
- Duplicate filtering with configurable timeout
- Signal strength (RSSI) capturing when available

## [0.9.0] - 2025-03-01
### Added
- Beta release for testing
- Core packet capture functionality
- Basic command line interface
